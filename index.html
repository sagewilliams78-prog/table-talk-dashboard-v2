How to send as a link

  1. Upload this HTML file to Netlify Drop, GitHub Pages, or another static host.
  2. Send the generated public link to the client.
  3. The client opens the link and edits data in the "Edit Products" tab.
  4. Edits save in that browser using local storage. Use Backup Data to transfer data between devices.
